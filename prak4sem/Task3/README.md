Шевченко Глеб, 212 группа

## Описание задания

Данное задание реализует обработку текстового ввода, в котором строки разбиваются на отдельные слова и специальные символы. Код включает следующие этапы выполнения:

1. **Чтение входной строки**: Считываются строки из стандартного ввода (`stdin`) до достижения конца файла (EOF).
2. **Разделение строки на слова и специальные символы**: Программа разделяет входные строки на слова и специальные символы (такие как `|`, `&`, `;`, `>`, `<`, `(`, `)`), а также учитывает двойные специальные символы (`||`, `&&`, `>>`).
3. **Сохранение слов в список**: Каждая лексема сохраняется в динамически растущем списке.
4. **Сортировка слов**: Лексемы сортируются в лексикографическом порядке.
5. **Вывод результата**: Результаты выводятся на экран, включая исходный и отсортированный списки слов.

## Основные функции программы

- **`main()`**: Основной цикл программы, который считывает входные строки, обрабатывает их и выводит результаты.
- **`addsym(char c)`**: Добавляет символ в буфер текущего слова, расширяя его при необходимости. Если текущий буфер заполнен, функция увеличивает его размер с помощью `realloc()`. Проверка успешного выделения памяти необходима, чтобы предотвратить ошибки и утечки. Функция также обновляет индекс текущей позиции в буфере (`curbuf`) после добавления символа.
- **`addword()`**: Добавляет слово из буфера в список слов и освобождает буфер для следующего слова. Эта функция завершает текущее слово, добавляя символ окончания строки (`\0`), и затем выделяет память для сохранения слова в списке. Если текущий список заполнен, функция увеличивает его размер с помощью `realloc()`, чтобы вместить новое слово. После добавления слово сохраняется в списке (`lst`), а буфер обнуляется для последующего использования.
- **`sortlist()`**: Сортирует список слов в лексикографическом порядке.
- **`printlist()`**: Печатает текущий список слов на экран.

## Компиляция и запуск программы

Для компиляции программы используйте команду:

```sh
gcc t1.c -o a
```

Для запуска программы выполните следующую команду:

```sh
./a
```

Затем введите строки для обработки. Для завершения ввода используйте комбинацию клавиш `Ctrl+D`.

## Пример использования

Программа считывает входной текст и сегментирует его на слова и специальные символы. Например, при вводе следующей строки:

```
command1&&command2>file.txt
```

Программа разобьет её на следующие элементы:

- `command1`
- `&&`
- `command2`
- `>`
- `file.txt`

Затем эти элементы будут отсортированы и выведены на экран.
