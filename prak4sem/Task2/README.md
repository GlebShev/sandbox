# README

**Шевченко Глеб**, 212 группа.

## Описание

В данном разделе содержатся решения задач **task2**, включая дополнительный пункт. 

## Задачи

### 1. Итеративное вычисление квадратного корня (task 2.1)
- На вход подается действительное число ε, а затем последовательность чисел до конца файла.
- Для каждого числа вычисляется квадратный корень с использованием метода Ньютона и выводится результат, пока разница между итерациями не станет меньше ε.

### 2. Вычисление многочленов (task 2.2)
- **task 2.2.1**: Для последовательности коэффициентов многочлена и числа x вычисляется значение многочлена по схеме Горнера.
- **task 2.2.2**: Дополнительно вычисляется производная многочлена в точке x.
- **task 2.2.3**: Дополнительно вычисляется определенный интеграл многочлена от 0 до x.

### 3. Последовательность Фибоначчи (task 2.3)
- Описаны итеративная и рекурсивная версии вычисления i-го члена последовательности Фибоначчи. Программа выводит оба варианта для введенного числа и сравнивает время их выполнения.

### 4. Преобразование строки в число (task 2.4)
- Функция `str2double()` преобразует строку в действительное число типа double без использования стандартных функций.

### 5. Работа со списком слов (task 2.5)
- Реализован список слов, который вводится с клавиатуры. Программа удаляет дубликаты слов, оставляя последнее слово, и выводит результат.

### 6. Дерево поиска (task 2.6)
- Реализовано дерево поиска с поддержкой операций вставки, удаления и поиска элементов.

### 7. Калькулятор выражений (task 2.7)
- Реализован калькулятор, который поддерживает операции сложения, умножения, скобок. Расширен поддержкой операций вычитания, деления и возведения в степень.

## Запуск
Для каждой задачи следует компилировать и запускать соответствующую программу с вводом данных через стандартные потоки.

```bash
gcc task7.c -o a
./a
```
