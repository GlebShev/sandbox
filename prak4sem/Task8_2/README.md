# Task 8_2

**Шевченко Глеб, группа 212**

**Вариант:** 4. Вещественная матрица, Вариант 4.1


## Постановка задачи

Необходимо реализовать абстрактный тип данных, представляющий вещественную матрицу и возможности работы с ней. Требуется обеспечить:
- Реализацию различных конструкторов (создание матрицы заданного размера, матрицы-строки, матрицы-столбца, создание из строкового представления)
- Управление динамической памятью (конструктор копирования, оператор присваивания, деструктор)
- Перегрузку операторов доступа (оператор [] для получения строк и элементов), скалярного умножения, оператора вывода
- Обработка ошибок с выбрасыванием исключений через вложенный класс


## Реализация

В проекте реализованы следующие компоненты:
- **matrix.h** – заголовочный файл, содержащий объявление класса `matrix` с необходимыми конструкторами, методами доступа, перегрузками операторов и вложенным классом исключений
- **matrix.cpp** – файл с реализациями всех методов класса: конструкторы, конструктор копирования, оператор присваивания, деструктор, статические методы `identity` и `diagonal`, методы `set` и `get`, оператор `[]`, операторы скалярного умножения(`*` и `*=`) и оператор вывода(`<<`)
- **main.cpp** – тестовая программа, демонстрирующая создание матриц разными конструкторами, изменение элементов, глубокое копирование, использование оператора [] и скалярного умножения

Для обработки ошибок используется вложенный класс `matrix::error`, который наследуется от `std::runtime_error` и выбрасывается при некорректных параметрах или выходе за пределы индексов


## Запуск

Для сборки проекта используется Makefile:
- Запустите сборку проекта:
   ```
   make
   ```
   Это создаст исполняемый файл с именем `test`.

- Запустите программу:
   ```
   ./test
   ```

- Для очистки скомпилированных файлов выполните:
   ```
   make clean
   ```
