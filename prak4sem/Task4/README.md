## 1. Команда `echo`

Команда `echo` выводит переданный текст на стандартный вывод. Поддерживает флаг `-n`, который отключает добавление символа новой строки в конце.

### Использование:

```
echo [-n] текст
```

- **-n**: не добавлять символ новой строки в конце вывода.

Пример:

```
echo "Hello, world!"
```

Вывод:

```
Hello, world!
```

## 2. Команда `pwd`

Команда `pwd` выводит текущую рабочую директорию.

### Использование:

```
pwd
```

Пример:

```
pwd
```

Вывод:

```
/Users/glebsevcenko/task4
```

## 3. Команда `ls`

Команда `ls` выводит содержимое текущего или указанного каталога. Поддерживает несколько флагов.

### Использование:

```
ls [-R] [-l] [-g] путь
```

- **-R**: рекурсивно выводить содержимое всех подкаталогов.
- **-l**: выводить детальную информацию о файлах, включая тип, права доступа, владельца и размер.
- **-g**: выводить имя группы владельца.

Пример:

```
ls -l -g -R echo.c
```

## 4. Команда `ln`

Команда `ln` создает ссылки на файлы. Поддерживает создание жёстких и символических ссылок.

### Использование:

```
ln [-s] original_link new_link
```

- **-s**: создать символическую ссылку (по умолчанию создаётся жёсткая ссылка).

Пример создания символической ссылки:

```
ln -s f1 f2
```

## 5. Команда `wc`

Команда `wc` подсчитывает количество строк, слов и символов в файле.

### Использование:

```
wc файл
```

Выводит имя файла, количество строк, слов и символов.

Пример:

```
wc file.txt
```

Вывод:

```
10 50 200 file.txt
```

(где 10 — количество строк, 50 — количество слов, 200 — количество символов)

## 6. Команда `cat`

Команда `cat` выводит содержимое файлов. Поддерживает флаг для нумерации строк.

### Использование:

```
cat [-n] файл1 [файл2 ... файлN]
```

- **-n**: выводить строки с номерами.

Пример вывода с нумерацией строк:

```
cat -n file1.txt file2.txt
```

Вывод:

```
     1  Первое содержимое файла
     2  Второе содержимое файла
```

## Как использовать

Для компиляции всех программ можно использовать `Makefile`, который создаёт исполняемые файлы для каждой из программ. Выполните команду:

```
make
```
