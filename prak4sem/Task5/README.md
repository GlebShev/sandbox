### Task 5 Шевченко Глеб 212

Данный проект реализует  интерпретатор команд (shell) на базе операционной системы Unix. 

**Возможности:**
- Приём пользовательских команд, их лексический и синтаксический анализ, выполнение.
- Поддержка внешних и внутренних команд (`cd`, `pwd`, `exit`).
- Поддержка конвейеров (`|`), перенаправления ввода-вывода (`<`, `>`, `>>`), фонового режима (`&`).
- Логические операции `&&` и `||`, последовательное выполнение `;`.
- Поддержка подшеллов `( ... )`.
- Подстановка переменных `$HOME`, `$USER`, `$SHELL`, `$EUID`.

### Структура проекта

- **list.c / list.h**:  
  Лексический анализ (разбиение строки на лексемы).  
  Реализует динамическое чтение строки `read_line()`, разбор кавычек, экранирования, комментариев, подстановки переменных окружения и создание/очистку списка лексем.
  
- **tree.c / tree.h**:  
  Синтаксический анализ с помощью рекурсивного спуска.  
  Превращает список лексем в синтаксическое дерево, представляющее структуру команды с приоритетами операторов.

- **exec.c / exec.h**:  
  Выполнение полученного синтаксического дерева.  
  Обход дерева, запуск команд через `fork()` и `execvp()`, обработка внутренних команд, конвейеров, перенаправлений, фонового режима, логических операторов.

- **main.c**:  
  Точка входа.  
  Организует диалог с пользователем: выдает приглашение, считывает команду, выполняет последовательность: лексический анализ → синтаксический анализ → выполнение.  
  Завершается при вводе команды `exit` или при достижении EOF (`Ctrl+D`).

### Сборка

В корне проекта имеется `Makefile`. Для сборки проекта выполните:

```bash
make
```

Это создаст исполняемый файл `my_shell`.

Для очистки объектных файлов и бинарных результатов:

```bash
make clean
```

### Запуск

Запустить полученный шелл можно командой:

```bash
./my_shell
```

После этого вы увидите приглашение `=> `, и можете вводить команды, как в обычном шелле:

Примеры:

- Запуск простой команды:
  ```bash
  => ls
  ```
  
- Конвейер:
  ```bash
  => ls | wc -l
  ```
  
- Перенаправление вывода:
  ```bash
  => ls > out.txt
  ```

- Логические операции:
  ```bash
  => ls && pwd
  => false || echo "Command failed, but we proceed"
  ```

- Подшелл:
  ```bash
  => ( cd ..; ls ) | sort
  ```

- Фоновый режим:
  ```bash
  => sleep 5 & echo "This prints immediately"
  ```

Ввести `exit` для завершения работы:

```bash
=> exit
```
